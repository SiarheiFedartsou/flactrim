flactrim Homepage

Description

flactrim is a program that allows "to cut" multitrack FLAC files on the separate files(using CUE sheets). It is licensed under the GPL. flactrim is developed in C++ under the Linux platform. But, I think, no difficulty to compile program under the Windows or MacOS platform. If you want to do it, do it and let me know the results. If you want to help me in developing, connect with me by e-mail.
